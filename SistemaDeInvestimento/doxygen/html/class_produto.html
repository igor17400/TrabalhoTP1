Detailed Description

Classe criada para representar a entidade Produto.

Descrição

Essa classe entidade é composta por outras classes Domínios. Por isso será expressado aqui todas as regras de cada domínio.

Requisitos dos atributos presentes nessa classe:

Classe CodigoProduto

Formato XXX onde cada X é dígito (0 – 9) e o código não pode ser 000.

Classe do tipo Classe

Texto CDB, LCA, LCI, LF ou LC.

Classe Emissor

Formato XXXXXX onde cada X é dígito (0 – 9) e não há dígito repetido.

Classe Prazo

Valor 6, 12, 18, 24, 30, 36, 42, 48, 54, 60, 66 ou 72 (meses).

Classe Data

Formato DD/MM/AAAA onde DD é número de 1 e 31. MM é número de 1 e 12. AAAA é número de 2020 a 2099. O valor da data deve considerar a ocorrência de anos que sejam bissextos.

Classe Taxa

Valor na faixa de 0 a 200 (% ao ano).

Classe Horario

Formato XY:ZW para representar horário entre 13:00 e 17:00 horas.

Classe ValorMinimo

Valor 1.000,00, 5.000,00, 10.000 ou 50.000,00 (reais).
