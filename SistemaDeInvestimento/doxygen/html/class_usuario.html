Detailed Description

Classe criada para representar a entidade Usuário.

Descrição

Essa classe entidade é composta por outras classes Domínios. Por isso será expressado aqui todas as regras de cada domínio.

Requisitos de atributos dessa classe:

Classe Nome

Texto composto por 5 a 30 caracteres onde cada caracter pode ser letra (A-Z ou a-z) ou espaço, há pelo menos 5 letras, não há espaços em sequência, primeira letra de cada termo é letra maiúscula.

Classe CPF

Formato XXX.XXX.XXX-XX onde cada X é dígito (0 – 9) e cada CPF inclui dígitos verificadores calculados por meio de algorítmo apropriado.

Classe Senha

Formato XXXXXX onde cada X é dígito (0 – 9) e não há dígito repetido.

Classe Endereço

Texto composto por 5 a 20 caracteres onde cada caracter pode ser letra (A-Z ou a-z), dígito (0 – 9), ponto ou espaço. Apenas letras e dígitos podem estar em sequência. Em termo cujo primeiro caracter é letra, a mesma é maiúscula.

Classe CEP

cep - Deve possuir o valor nas seguintes faixas: 1000000 a 5999999 e 8000000 a 8499999 (São Paulo), 20000000 a 26600999 (Rio de Janeiro), 70000000 a 70999999 (Brasília), 40000000 a 41999999 (Salvador) e 60000000 a 60999999 (Fortaleza).
